2. Append to `app/actions/admin.ts`:

**Events Actions:**

a. `getEventsAction(filter: 'upcoming' | 'past' = 'upcoming')`:
   - requireAdmin().
   - Query supabaseAdmin.from('events').select('*, locations(name)').
   - Filter: upcoming = event_date >= now(), past = event_date < now().
   - Order by event_date (ascending for upcoming, descending for past).
   - Return events array.

b. `createEventAction(formData: FormData)`:
   - requireAdmin().
   - Extract: title_es, title_en, description_es, description_en, event_date, event_type, start_time, end_time, image_url, location_id.
   - Validate: title_es and title_en required, event_date required, event_type must be tournament/training/social.
   - Insert into events table via supabaseAdmin.
   - Return { success: true }.

c. `updateEventAction(eventId: string, formData: FormData)`:
   - requireAdmin().
   - Same field extraction as create.
   - supabaseAdmin.from('events').update({...}).eq('id', eventId).
   - Return { success: true }.

d. `deleteEventAction(eventId: string)`:
   - requireAdmin().
   - supabaseAdmin.from('events').delete().eq('id', eventId).
   - Return { success: true }.

**CMS Actions:**

e. `getContentBlocksAction()`:
   - requireAdmin().
   - supabaseAdmin.from('content_blocks').select('*').order('sort_order', { ascending: true }).
   - Group by page prefix (home_, about_, learn_, faq_) on the server side.
   - Return grouped blocks: { home: ContentBlock[], about: ContentBlock[], learn: ContentBlock[], faq: ContentBlock[] }.

f. `updateContentBlockAction(blockId: string, content_es: string, content_en: string)`:
   - requireAdmin().
   - supabaseAdmin.from('content_blocks').update({ content_es, content_en, updated_at: new Date().toISOString() }).eq('id', blockId).
   - Call revalidatePath('/') to trigger ISR cache invalidation so Phase 5 public pages see fresh content.
   - Return { success: true }.

g. `reorderContentBlocksAction(blockIds: string[])`:
   - requireAdmin().
   - For each blockId at index i: supabaseAdmin.from('content_blocks').update({ sort_order: i + 1 }).eq('id', blockIds[i]).
   - Return { success: true }.

Update i18n messages (append to Admin namespace):
- Events: events, createEvent, editEvent, deleteEvent, eventTitle, eventTitleEs, eventTitleEn, descriptionEs, descriptionEn, eventDate, eventType, tournament, training, social, startTime, endTime, imageUrl, upcomingEvents, pastEvents, noEvents, confirmDeleteEvent, confirmDeleteEventMessage, eventCreated, eventUpdated, eventDeleted
- CMS: contentBlocks, homeBlocks, aboutBlocks, learnBlocks, faqBlocks, editContent, spanishContent, englishContent, preview, saveChanges, saved, moveUp, moveDown, lastUpdated
- Stripe: stripePayments, stripeDescription, openStripeDashboard

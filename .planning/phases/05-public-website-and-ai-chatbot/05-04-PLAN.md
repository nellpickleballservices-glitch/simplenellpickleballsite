    Rewrite MobileNav.tsx to be a dropdown menu instead of a slide-out drawer. Keep the same props interface and all existing navigation links/auth buttons.

    **Changes required (per debug diagnosis):**

    1. **Hamburger button**: Change from open-only to toggle (`onClick={() => setOpen(!open)}`). Remove the 3-line hamburger when open and show an X icon instead, or use a simple toggle. Keep `md:hidden`.

    2. **Remove body scroll lock**: Delete the entire `useEffect` that sets `document.body.style.overflow`. A dropdown does not need scroll lock.

    3. **Remove backdrop overlay**: Delete the full-screen `m.div` with `fixed inset-0 bg-black/60`. Instead, add a click-outside handler using a `useEffect` with `document.addEventListener('mousedown', ...)` to close the dropdown when clicking outside.

    4. **Change panel positioning**: Replace `fixed top-0 right-0 z-50 h-full w-72` with `absolute top-full right-0 z-50 w-64` (or `w-full`). This positions the dropdown below the navbar (using `top-full` relative to the sticky nav parent). Add `rounded-b-lg` or similar for visual polish.

    5. **Change animation**: Replace x-axis slide (`x: '100%'` to `x: 0`) with y-axis or opacity animation:
       ```
       initial={{ opacity: 0, y: -8 }}
       animate={{ opacity: 1, y: 0 }}
       exit={{ opacity: 0, y: -8 }}
       transition={{ duration: 0.15 }}
       ```

    6. **Remove close button**: Delete the dedicated X close button `div` at the top of the panel. The hamburger toggle handles open/close.

    7. **Simplify layout**: Remove `flex-col` full-height layout with `mt-auto`. Stack all items naturally: nav links first, then language switcher, then auth buttons. Use `py-2 px-4` padding. Keep existing Tailwind color classes (`bg-midnight`, `border-charcoal`, etc.).

    8. **Keep all existing functionality**: All public links (about, learn-pickleball, events, contact, pricing), authenticated links (reservations, dashboard, admin), language switcher, login/signup buttons, and logout form must remain. Each link must still call `close` on click.

    Keep `AnimatePresence` and `m.div` from `motion/react` for the dropdown animation. Keep all `useTranslations` hooks and the `Link` imports.

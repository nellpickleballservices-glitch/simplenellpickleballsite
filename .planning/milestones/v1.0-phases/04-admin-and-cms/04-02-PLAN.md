Create `PricingGrid.tsx` ('use client'):
- Receives `initialData: CourtPricingGrid[]` prop
- Displays a table/grid for each court showing 7 columns (Mon-Sun, reordered so Monday is first for business convention) with the price in dollars (price_cents / 100)
- Each cell is editable: click to switch to an input field, type new price in dollars, blur or Enter to save
- On save, call `upsertSessionPricingAction(courtId, dayOfWeek, priceCents)` where priceCents = Math.round(dollars * 100)
- Show a brief success indicator (green checkmark or flash) on save
- Show error toast/message if save fails
- Use locale-aware day names from DAY_NAMES_EN/DAY_NAMES_ES based on current locale (pass locale as prop or use useLocale from next-intl)
- Styling: dark table with border-gray-700 borders, hover highlights, lime accent on active edit cell

Create `SurchargeEditor.tsx` ('use client'):
- Receives `initialValue: number` prop (e.g. 25 for 25%)
- Displays current surcharge with a percentage sign
- Editable input (number type, min 0, max 100, step 1)
- Save button that calls `updateTouristSurchargeAction(pct)`
- Show success/error feedback
- Brief explanation text: "Applied to all non-Dominican users" / "Se aplica a todos los usuarios no dominicanos"
